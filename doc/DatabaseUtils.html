(static) remove(model, primaryKey, object, callback)

If the object already exists in the model, remove that record from the database, and then call the callback on the successful response. Otherwise, call the callback on the failure response. Note that the primary key is supposed to be unique, so there should be at most 1 such document found in a specific collection.
